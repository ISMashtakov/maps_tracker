# Приложение для отслеживания позиции

## Требования

### Общие требования
- Нижняя панель навигации автоматически скрывается.

### Экран с главным меню
- При входе в приложение открывается главное меню. 
- Меню состоит из кнопок:
    - Кнопка загрузки уровня.
        - При нажатии открывается окно выбора файла на устройстве.
        - После выбора файл добавляется в список кнопок ниже.
        - Выбранные файлы сохраняются от запуска к запуску.
    - Список кнопок с приключениями, который может быть больше, чем поместиться на экране. Предусмотреть прокрутку.
        - Список состоит из файлов загружженых при помощи кнопки выше. 
        - В списке только уникальные элементы. При попытке загрузить файл повторно, выдавать предупреждение "Файл уже загружен" и не загружать его.
        - Имеется кнопка для удаления элемента.
        - При нажатии по кнопке открывается основной экран.

### Основной экран
- Постоянно показывать карту и позицию пользователя на ней.
